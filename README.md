# Econis — Organic E-Commerce Landing Page

Premium organic / healthy drinks landing page built with React, Vite, Tailwind CSS, Framer Motion, and React Icons.

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- Framer Motion
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI (Navbar, ProductCard, etc.)
├── sections/       # Page sections (Hero, Categories, etc.)
├── data/           # Dummy JSON data
├── pages/          # HomePage
├── assets/
├── App.jsx
└── main.jsx
```

## Features

- Sticky transparent → solid navbar on scroll
- Hero with floating bottles, leaves, juice splash
- Hot categories horizontal scroll / grid
- Dual promo banners
- Top products grid with lazy images & skeleton loading
- Responsive (mobile / tablet / desktop)
- Glassmorphism, soft gradients, micro-interactions
